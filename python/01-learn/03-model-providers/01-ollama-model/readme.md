# Directory Overview

This directory contains:

- `sample_file/`: Sample files, including the Amazon 2023 Shareholder Letter PDF.
- `images/`: (Placeholder) Directory for image assets.
- `ollama_file_ops_agent.ipynb`: Jupyter notebook demonstrating file operations with the Ollama agent.
- `requirements.txt`: Python dependencies required for the notebook.
- `sample.txt`: A test file created by the Ollama agent.
- `readme.md`: This file.

## Purpose

The repository is intended for learning and experimenting with file operations using an Ollama-powered agent.

## Usage

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Launch the Jupyter notebook:
   ```bash
   jupyter notebook ollama_file_ops_agent.ipynb
   ```
3. Follow the notebook instructions to read, write, list directories, and summarize files.
